Sequence Viewer

SequenceViewer is a package to use in order to highlight specific region of a protein or of aligned sequences
SequenceViewer: "viewSequence" mode

Usage

  • mode: viewSequence mode is used to display information about a sequence. In this mode, the input must be a single sequence.
  • view: Display characters or images on top of the sequence. In sequence viewer each element of the view array is intended as a row of residues, numbers, symbols or images. In this way, also secondary structure or post-transcriptional modification can be indicated.
  • parameters: Show a space between chunks of sequence: You can set the length of chunks (number of residues per chunk) using the parameter chunkSize. If you want a fixed layout set the desired chunkSize and set the width of the parent div. Choose the desired fontSize. (pixel or vw for fixed layouts). If not set, default value will be 14px. SkipIndex sets the number of alternating indexes to show. Particular cases: If skipIndex is not set, no index will be showed. Set skipIndex to the sequence length to display all indexes instead. It is possible to set the colorChoice in 3 different modes. adjacent: color regions with adjacent colors of the color spectrum, opposite: color regions with opposite colors of the color spectrum, custom: allows the user to choose his own colors for regions. In this case, besides setting the colorChoice to customizable, colors has to be set with the specific colors option. It is necessary to insert a color for each region.
  • colors: Specify type, bounds (start, end) and colors. For type “insertion” a color must always be set. For regions instead, you may choose a color through the custom color option or not, setting one of the default modes (adjacent or opposite).
